# 🏴‍☠️ Captain Byte's Treasure Quest — v1 (CLI)

The Volume I deliverable: a short, funny, complete text adventure.

## How to run (once built)

```bash
python main.py
```

- Runs start to finish, no crashes on expected input.
- A kid who didn't write it can play it and smile.
- Uses only Volume I skills.
- The child can honestly say: **"I built this."**
