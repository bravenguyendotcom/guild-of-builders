# 💻 source_code/

The **actual game**, one folder per volume. The same game evolves; each volume gets its own
snapshot so the child can see how far they've come.

```
source_code/
├── volume_1/treasure_quest/   # CLI version (Volume I deliverable)
├── volume_2/treasure_quest/   # evolved (later)
├── volume_3/treasure_quest/   # published GUI/web (later)
└── volume_4/treasure_quest/   # AI-powered (later)
```

Spec: `docs/GAME_DESIGN.md`.
