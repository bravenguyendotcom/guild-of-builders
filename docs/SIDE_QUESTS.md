# 🧭 SIDE_QUESTS.md

### Optional adventures — for Builders who want more

> **Status:** DRAFT.
> **What this is:** the home for **optional** challenges that sit *beside* the main voyage —
> never required to finish the series. Perfect for a fast Builder who'd otherwise get bored
> (see `BUILDERS_LOGBOOK.md`, the Fast Track).
> **The scope rule (D-19 & the Treasure Chest):** the **core spine stays the one game.**
> Side-quests are extras and experiments — they must never become required, and must never
> pull focus from *Captain Byte's Treasure Quest.*
> **The IP rule (D-19):** everything here is an **original creation.** We take inspiration,
> then give it our own Python-Planet skin. No borrowed characters.

---

## How side-quests work

- **Always optional.** A Builder can skip every one and still reach 🏛️ Guild Builder.
- **North Star still applies.** A side-quest earns its place only if it makes a better Builder.
- **They reward, they don't gate.** Finishing one earns a badge or bragging rights — never a
  key you *need* to continue the main story.
- **Great for the Fast Track.** Spare speed becomes more building, not boredom.

## ⚔️ Featured Side-Quest — The Typing Dojo 🥋 (with The Tangle 🦑)

The Guild's arcade: Builders train by **typing real, runnable Python** — and get quietly hooked
on Computer Science along the way. Its guardian is **The Tangle**, a ten-tentacled Typo-Squid
(our own creature — D-19) who feeds on sloppy typing.

**It lives in its own home:** [`typing_dojo/`](../typing_dojo/) — see
[`typing_dojo/README.md`](../typing_dojo/README.md) for the full spec (engine + skin
architecture, the four-tier ladder, the frozen mission format, and Batch 01).

The taste, so this index stands on its own:

- **We build no typing app.** Free tools already exist (10FastFingers, keybr, monkeytype,
  typing.com — several have a code mode or file upload). We supply the *practice files*, not the app.
- **🥋 Challenge I — Passive Memory:** type a whole real CLI program, run it, hunt your own typos.
  Syntax seeps into your fingers without trying to memorize a thing.
- **🕵️ Challenge II — Detective Mode:** retype code seeded with **typos only** (never logic bugs),
  spotting and fixing as you go. *"Excellent. Now the conversation begins."*
- **Badge:** 🥋 **Dojo Belt** — for cleaning a Challenge file until it runs, at your own pace (D-15).

*(Full canon for the Dojo — the four tiers, engine/skin, repetition strategy — is locked in
`DECISIONS.md` D-20–D-23 and specced in `typing_dojo/README.md`. This entry is just the index card.)*

## 🗺️ The Side-Quest Board (ideas parked for later)

Optional extras a hungry Builder can explore. Each stays a *side*-quest — never core spine.

- **Speed-Runner's Challenges** — the Bonus ⭐ tasks from missions, collected in one place.
- **Build-Your-Own** — after each volume, invent a small original program of your own before
  sailing on. (The Fast Track's favourite.)
- **Explorer's Extras** *(for older/hungrier Builders like Tommy, someday):* short, optional
  tastes of the wider world of Python — automation, data, or simple AI experiments. Kept
  **strictly optional and off the main spine** (the boundary D-19 & the Treasure Chest protect).
  These are "if you're curious" doors, not required rooms.
- **Pycasso's Gallery** 🎨 — optional art challenges: make your program's output beautiful
  (ASCII art, colour, a title screen worth showing off).
- **Riddle Theatre Open-Mic** 🎭 — write your *own* riddle or joke for the Guild board, in the
  spirit of Maestro Monty & Sir Quizzalot. (Tommy's 7-8-9 riddle started it all.)

## Adding a new side-quest (checklist)

Before adding one, confirm:

- [ ] It's genuinely **optional** (skipping it never blocks the main story).
- [ ] It passes the **North Star** (makes a better Builder).
- [ ] It's an **original creation** (D-19 — no borrowed IP).
- [ ] It uses skills the Builder **already has** at that point in the series.
- [ ] It rewards **personal progress**, not competition between kids (D-15).

want to — and come back from a slightly stronger Builder.*
