# ✅ workbook/

`docs/STYLE_GUIDE.md` (Story → Mission → Requirements → Example I/O → Think First →
Planning → Coding → Testing table → Debug log → Reflection → Bonus ⭐ → Difficulty).

```
workbook/
└── volume_1/
    ├── 01_hungry_pizza_robot.md
    ├── 02_the_age_machine.md
    └── ...
```
