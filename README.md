# 🏴‍☠️ The Guild of Builders

> *An open-source adventure that teaches children to build software, games, AI —
> and themselves.*
>
> **We are not teaching Python. We are raising Builders.**

Made by **Brave (Dad)**, with love, for **Tommy and Teppy** — and for any curious
10–15 year old who wants to build things. ❤️

---

## What is this?

A story-driven learning series where children learn Python, software engineering, AI, and
computational thinking by building **one evolving game across four volumes**:

### 🎮 *Captain Byte's Treasure Quest*

| Volume | The child builds... | And can say... |
|--------|--------------------|----------------|
| **I — The First Voyage** | a playable CLI adventure | "I built a game." |
| **II — Growing the Adventure** | the same game, evolved | "I made it bigger." |
| **III — The Living World** | a published GUI/web game | "People can play my game." |
| **IV — The Infinite Guild** | an AI-powered game & apps | "I build *with* AI." |

The game is **never restarted** — it grows with the Builder. That's the whole idea.

## The two rules that keep this alive

1. **One artifact at a time.** Edit one file per session; ship a new version; commit it.
2. **The North Star:** *"Will this help a child become a better Builder?"*

And the running law we never break — **the Donut Law** 🍩: every session ends with a real,
delivered file. No promises.
