# 📖 manuscript/

The **story** the child reads — the adventure novel that carries the lessons.

Organize by volume, one chapter per file:

```
manuscript/
└── volume_1/
    ├── 00_welcome.md
    ├── 01_the_pizza_robot.md
    ├── 02_the_first_bug.md
    └── ...
```

**Rules:** story first, Python second (`docs/STYLE_GUIDE.md`); characters stay in voice
(`docs/CHARACTERS.md`); one chapter per session (the Donut Law 🍩).
